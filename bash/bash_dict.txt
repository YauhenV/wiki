cat - Самый простой способ прочитать файл.
cd (change directory) - Осуществляется перемещение по файловой структуре. Для этого ей нужно передать аргумент — директорию, в которую необходимо переместиться.
cd [file_name] [name_file_copy] - копирует файл. Её первый аргумент — имя источника (откуда), второй — имя приёмника (куда). 
chmod (от англ. change mode) — команда для изменения прав доступа к файлам и каталогам, используемая в Unix-подобных операционных системах.
chown (от англ. change owner) — UNIX‐утилита, изменяющая владельца и/или группу для указанных файлов.
clear - очищает окно терминала.
