A
a (anchor)(тэг. ссылки) - является одним из выжных элементов HTML и предназначен для создания ссылок.
action (атрибут form) - указывает обработчик, к которому обращаются данные формы при из отправки на сервер.
address  (тэг. семантика) - определяет блок с контактной информацией, относящейся к ближайшему <article> или <body>.
article (тэг. семантика) - обозначает законченную самодостаточную часть документа: пост на форуме или в блоге, новостная статья или виджет.
aside  (тэг. семантика) - определяет блок с сопуствующим содержимым (для обозначения выносок или цитат).
audio (тэг. вставка) - добавляет, воспроизводит и управляет настройками аудиозаписи на веб-странице.

B
base (тэг.указывается в head) - определяет базовый адрес для всех ссылок документа, а также как они будут открываться.
blockquote (тэг) - предназначен для выделения длинных цитат внутри документа.
br (break line)(тэг. текст) - устанавливает перевод строки в том месте, где он находится.
button (тег. формы) - создаёт на веб-странице кнопки.

C
caption (тэг. таблица) - Создаёт заголовок таблицы. 
class (универсальный атрибут) - задаёт класс элементов, позволяя группировать их.
code (тэг. текст) - предназначен для отображения одной или нескольких строк текста, который представляет собой программный код.
colspan (тэг. таблица) - объединяет ячейки по горизонтали (в строчку).

D
disabled (атрибут input) - блокирует доступ и изменения элемента.
div (division)(тэг) - универсальный группирующий элемент для изменения вида содержимого через стили. 

E
em (тэг. текст) - предназначен для акцентрирования текста (курсив).

F 
figcaption (тэг) - для добавления подписи к элементам figure 
figure (тэг) - используется для группировки любых элементов
footer  (тэг. семантика) - определяет подвал всего документа или подвал секции.
form (тег. формы) - предназначена для обмена данными между пользователем и сервером.

G

H
h1-h6  (тэг. семантика) - определяет заголовок для секционных элементов или <body>. Число определяет уровень: 1 - наивысший, 6 - наименьший.
header  (тэг. семантика) -  определяет шапку всего документа или шапку секции.
hgroup  (тэг. семантика) - определяет группу заголовков.
hr (horizontal rule)(тэг) - горизонтальная линия для тематического разделения параграфов.
href (атрибут тэга a) - задаёт адрес документа, на который следует перейти.

I
id(универсальный атрибут) - задаёт уникальный идентификатор элемента.
iframe (тэг. вставка) - позволяет загружать в область заданных размеров любые другие независимые документы.
img (тэг) - предназначен для отображения не веб-странице изображений в графическом формате (GIF,JPEG,SVG,PNG).
input (тэг. формы) - позволяет создать разные части интерфейса и обеспечивать взаимодействие с пользователем.

J

K
kbd (keyboard)(тэг. текст) - используется для обозначения текста, который набирается на клавиатуре или для названия клавиш(моноширинный шрифт).

L 
label (тэг. формы) - устанавливает связь между определённой меткой, в качестве которой обычно выступает текст, и элементом формы (input, textarea, select).
li (list item)(тэг списка) - определяет элемент списка. 
link (тэг.указывается в head) - устанавливает связь с внешним документом, например с CSS файлом (где хранятся стили).

M
main (тэг. грипирующий элемент) - главное содержимое страницы.
mark (тэг. текст) - помечает текст как выделенный (закрашивает фон в желтый цвет).
media (атрибут тэга link) - описывает устройство, для которого следует применять стилевое оформление.
meta (тэг.указывается в head) - определяет метаинформацию документа.
method (тег. формы) - сообщает серверу о типе запроса.
multiple (атрибут select) - позволяет одновременно выбирать сразу несколько элементов списка.

N
nav  (тэг. семантика) - определяет блок с навигацией по связанным документам. Рекомендуется использовать только для главная навигации.

O
ol (ordered list)(тэг списка) - определяет упорядоченный список (1,2,3).

P
p (тэг. грипирующий элемент) - определяет параграф текста.
placeholder (атрибут input) - выводит текст внутри поля формы, который исчезает при получении фокуса или при наборе текста.
pre (preformatted text)(тэг) - определяет блок предварительно форматированного текста. Такой текст отображается моноширинным шрифтом, сохраняя всё форматирование: пробелы, отступы.

Q
q (quote)(тэг. текст) - предназначен для выделения цитат или обозначения прямой речи (текст в ковычках).

R
rowspan (тэг. таблица) - объединяет ячейки по вертикале(в высоту).
rel (атрибут тэга link) - описывает тип связи.
rubby, rt (тэг. текст) - предназначен для добавления небольшой аннотации сверху или снизу от заданного текста (текст выше основного контента).

S
samp (sample output) (тэг. текст) - используется для отображения текста, который является результатом вывода компьютерной программы или скрипта (моноширинный шрифт). 
script (тэг.указывается в head) - предназначен для описания скриптов, может содержать ссылку на скрипт или его текст на определённом языке.
section  (тэг. семантика) - определяет тематическую группу элементов: блок комментариев к посту в блоге, главы в статье или список постов.
select (тэг. формы) - элемент интерфейса в виде списка, позволяющий выбрать одно или несколько значений. 
small (тэг. текст) - предназначен для различной уточняющей информации. Например, для условий и правовых ограничений в рекламе(текст меньше).
span (тэг. текст) - универсальный текстовый элемент, который позволяет изменить вид содержимого через стили.
strong (тэг. текст) - предназначен для обозначения важной информации (Жирный).
style (тэг.указывается в head) - предназначен для определения стилей.
style (универсальный атрибут) - добавляет стили оформления (по правилам CSS) к элементу.
sub (subscript)(тэг. текст) - обозначает нижний индекс.
sup (superscript)(тэг. текст) - обозначает верхний индекс.

T
tabindex (тег. формы) - устанавливает порядок получения фокуса при переходе между элементами с помощью клавиш TAB
table (тэг. таблица) - создание таблицы
tbody (тэг. таблица) - обозначает данные таблицы.
td (тэг. таблица) - создание ячейки в таблице
textarea (тэг. формы) - представляет собой элементы формы для создания области, в которую можно вводить несколько строк текста.
tfoot (тэг. таблица) - обозначает подвал в таблице.
th (тэг. таблица) - создание ячейку шапки с названием столбцов.
thead (тэг. таблица) - обозначает шапку таблицы.
title (единственный обязательный тэг. указывается в head) - элемент для определения заголовка документа.
title (универсальный атрибут) - добавляет пояснения к содержимому элементу, которое появляется при наведении.
tr (тэг. таблица) - создание строки в таблице
track (тэг. вставка) - позволяет авторам указать текстовую дорожку для медийных элементов audio и video. 
type (атрибут input) - сообщает браузеру, к какому типу относится элемент форм. (password, text, file)

U
ul (unorder list)(тэг списка) - определяет неупорядочный список (маркированный).

V
video (тэг. вставка) - добавляет, воспроизводит и управляет настройками видеорилика на веб-странице.

W
wbr (word break opportunity)(тэг. текст) - указывает браузеру место, где допускается делать перенос строки в длинной слове.
