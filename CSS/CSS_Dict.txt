column-count - указывает нужное количество колонок в тексте
column-gap - меняет расстояние между колонками в тексте

column-width -  указывает ширину колонок

text-align - свойство для выравнивания текста
text-decoration - свойство для "Украшения" текста (подчеркнутый, зачеркнутый и подчеркнутый) 
text-indent - свойство для изминения величины отступа в абзаце.Допустимо использовать любые единицы измерения длины и их отрицательные значения.
text-overflow: ellipsis - скрывает переполнение текста (текст обрезан, но обрезанная часть заменена на три точки)
text-transform - трансформация текста, позволяет менять регистр букв в словах
transform - выполняется преобразование элемента (Перемещение, Масштабирование, Вращение, Наклон)
transform-origin - изменяется исходная точка при преобразовании элемента (можно указывать проценты, px и ключевые слова right, bottom. Исходная точка по умолчанию находится в центре элемента)
transform-style - сообщает о том что дочерние элементы в перспективе позиционируются в 3D-пространстве.
